# System Governance Admin Dashboard

A web-based admin dashboard for managing system governance, user roles, permissions, and operational metrics.

## Features

- Role-based access control (RBAC)
- User and team management
- Audit logs and activity tracking
- System health and metrics overview
- Policy and compliance management interface
- Responsive design for desktop and tablet

## Tech Stack

- **Frontend**: React / Next.js
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Auth**: JWT-based authentication

## Getting Started

```bash
git clone https://github.com/Salmanahmed1078/System-Governance-Admin-Dashboard.git
cd System-Governance-Admin-Dashboard
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.
